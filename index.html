Skills

HTML

This is the bread and butter of any web developer. I use this in all of my projects in one way or another. It works very well when building static web pages such as this one.

CSS

CSS is core at bringing style to any web page or application. I use CSS to style all my projects, including key frames, media queries, variables, and animations. I also love Flexbox!

Vanilla JavaScript

JavaScript is an incredibly powerful language used in today's browsers. In tandem with HTML and CSS, they make most of the web pages and applications used today.

React

React is my favorite JavaScript library. I use this tool for dynamic single-page applications. The way React uses reusable components comes in very handy. Depending on the size of the project I'm working on, I'll handle state differently. My regular go-to is regular prop usage or Context API. I'm currently learning Redux for bigger applications.

Node.js

I use Node for building the server-side portion of my projects. Node is great because it allows JavaScript to be used outside the browser, in the command line, and interact with the back end using only one programming language. I always pair this with the Express.js framework to create robust back-end architecture and APIs.

Databases

I normally use MongoDB paired with Mongoose and Atlas as a management tool in NoSQL projects where performance is key. This is my go-to to build MERN full-stack applications. Otherwise, I use SQLite3 with Knex and SQLite Studio as a management tool for SQL/RDBM applications.

Python

I know the basics of Python and have used it while I was learning the fundamentals of Computer Science. It is a strong and intuitive language that I mostly use to solve coding challenges and other complex problems.

Featured Projects

Below are some of my projects. I'm constantly building new applications and websites. All my work is on my GitHub page. Click here to access all my repositories.

E-Store

A project using the MERN stack. The user can create an account and add items to the cart. Checkout is easy and uses the PayPal SDK. If logged in as an admin, CRUD operations available to add new items to the Store and mark items as delivered. There is also a separate tab for admins to check order status and more features.

Beach Resort

A Beach Resort website. It uses Context API for state management. It has a reponsive design using media queries in CSS and a filtering system in real time using several JavaScript array methods. It has reusable React components such as the header and footer and uses Contentful to access the room data.

News Hub

News Hub is a news aggregator application that is controlled by voice commands using the Alan AI software. The AI will proceed to search for news depending on several voice commands it has been programmed to work with. It also reads article headlines and opens articles in a new tab when instructed to do so. Styling made with Material UI.

Game of Life

This is my take on Conway's Game of Life. The player can interact with the grid by clicking on it and new cells will be created. The player can also pre-load existing patterns and pause/reset/change game speed and grid size. There are also 'controls', 'rules', and 'about the game' sections. These are displayed using modals and styling from reactstrap.
